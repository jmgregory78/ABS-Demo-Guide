ZoomIt
Screen zoom and annotation tool for live demos. ✓▾ 🚩 Flag
ZoomIt is a free Microsoft Sysinternals tool that lets you zoom in on any part of the screen while demoing — without switching away from your application. It runs silently in the system tray and activates instantly with a hotkey. Practice it extensively before using it with a customer so it feels natural. 📥 Download from Microsoft Sysinternals ↗
📸 Visual Reference
Copilot Chat interface — shows the left nav areas (New chat, Search, Library, Create, Frontier) and pinned Agents panel. Click to enlarge.
One-Time Setup
1
Download and launch ZoomIt. Run the ZoomIt application (not ZoomIt64 or ZoomIt64a).
2
Open the Options menu — accessible from the system taskbar tray icon (click the caret to find it).
3
On the Zoom tab, configure:
A. Set Zoom Toggle to Ctrl + 2
B. Set zoom magnification level to 1.5 — optimal for Copilot demos
C. Check "Run ZoomIt when Windows starts"
4
On the LiveZoom tab, configure:
A. Set LiveZoom Toggle to Ctrl + 1 — easy to trigger with your left hand while demoing
B. Click OK
7
To exit ZoomIt, close it from the system tray — right-click the icon and select Exit.

7.3
Windows Clipboard History
Pre-load longer prompts into clipboard history for instant paste. ✓▾ 🚩 Flag
Windows Clipboard History lets you copy multiple strings at once and paste any of them on demand with Win+V. It's especially useful for long prompts that are too slow to type with Auto Hotkeys — pre-load them before your demo and paste instantly.
One-Time Setup — Enable Clipboard History
⚡ Quickest way: Press Win + V — if not enabled, Windows will show a prompt with a "Turn On" button. Click it and you're done. A
Alternative (via Settings): Settings → System → Clipboard → toggle Clipboard History on.
How to Use
1
Copy each prompt you need using Ctrl+C — copy them all before the demo starts.
2
Press Win+V in any text field to open the clipboard panel and see all copied items. B
3
Click any item to paste it, use the pin icon to keep favourites across restarts, or the … menu to delete individual items.
4
Clear all (button in top-right of the panel) before each demo to start with a clean history.
💡 Pro Tip: Use this for long prompts that are too slow for Auto Hotkeys — pin them in clipboard history so they survive restarts and are always ready.
